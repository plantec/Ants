## Getting Started

### Installing Ants

For Pharo 10 : 

```smalltalk
Metacello new
   baseline: 'Ants';
   repository: 'github://plantec/Ants';
   load.
```

### Starting an Anthill simulation

```smalltalk
Terrain start.
```
